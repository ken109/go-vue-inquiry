# go-vue-inquiry
## インストール
```
git clone https://github.com/ken109/go-vue-inquiry.git
cd go-vue-inquiry

docker-compose up
docker-compose down

docker-compse up
```

`docker-compose up`は二回行ってください。1回目起動時にmysqlの起動がapiサーバーの起動に間に合わなくでうまくapiサーバーが起動しないためです。
